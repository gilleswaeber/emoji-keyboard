The list can be generated with the following function: (open light.html in a browser and type this in the debug console)
``` Javascript
function getKeymaps(){
	r = '';
	_.each(data.keymaps, (a,k) => 
		r+= k + "\t| " +
			a.name + "\t| " +
			_.range(16,28).map((v) => a.keys[v]).join(" ") + "\n"
	);
	return r;
}
```
